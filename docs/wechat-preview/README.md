## 📝 发布步骤

1. 打开 [微信公众号后台](https://mp.weixin.qq.com)
2. 点击「创作管理」→「图文消息」→「新建」
3. 打开预览页面: index.html
4. 点击「复制文章」按钮
5. 在公众号编辑器中粘贴
6. 上传封面图
7. 检查排版 → 发布
